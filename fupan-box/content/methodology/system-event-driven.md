## 核心命题

> 事件驱动体系的核心是「**已知事件 + 未知反应 = 预期差**」。
> 你赚的不是事件本身, 而是**事件爆发后, 市场反应与你预期的差距**。
> 一句话: **市场已经知道的不值钱, 市场还没充分定价的才有钱**。

## 一、这个体系到底在做什么

A 股有几个**周期性出现的事件窗口**:

- **政策窗口**: 两会 (3 月) / 中央经济工作会议 (12 月) / 国常会 (每月) / 行业政策出台
- **业绩窗口**: 业绩预告期 (1 月、4 月、7 月、10 月) / 正式年报季 / 季报季
- **重组窗口**: 并购重组、资产注入、混改、分拆上市
- **生命周期窗口**: 解禁 / 限售股到期 / 高送转 / 现金分红
- **产业事件**: 新产品发布 (苹果发布会、华为新机、特斯拉投资者日) / 国际地缘事件

事件驱动交易者的工作是:

1. **提前预判事件可能发生**: 通过日历表 + 政策风向 + 公司公告 + 行业研究
2. **提前布局相关受益标的**: 在事件爆发前 1-4 周建仓
3. **事件爆发时观察市场反应**: 反应强 → 持有, 反应弱 / 利好出尽 → 立即清仓
4. **不在事件已经全市场知晓后才进场**: 那时已经是接最后一棒

> **「利好出尽是利空」「利空出尽是利好」**——这两句俗话, 就是事件驱动体系的灵魂。

## 二、目标周期与仓位结构

- **持有周期**: 数日 - 数月 (依事件类型而定)
- **仓位构成**: 同时持有 2-5 个事件主题, 单主题不超过 30%
- **目标年化**: 30-60% (事件密集年份可更高)
- **最大回撤接受度**: -15% 到 -20%
- **胜率/盈亏比目标**: 胜率 50-60%, 盈亏比 ≥ 1.5

## 三、适合什么人

适合:

- 关注新闻政策, 对宏观敏感
- 有产业资源 / 行业认知, 能预判事件影响
- 中等节奏 (不需全天盯盘, 但要每天看新闻)
- 资金量中等以上 (10-200 万)

不适合:

- 不关心新闻、不读研报的人
- 反应慢、信息处理速度跟不上市场的人
- 想做"高频接力"的人 (事件驱动周期更长)
- 不愿意接受"事件反应不及预期"亏损的人

## 四、核心买入逻辑 (事件分类 + 预期差判断)

### 第一类 政策驱动 (最大类)

举例:
- 「人工智能产业规划」即将发布 → 提前布局 AI 基础设施 / 算力 / 大模型
- 「住房政策放松」预期升温 → 提前布局地产链 / 家电 / 建材
- 「央国企改革」推进 → 提前布局相关央企

**关键技巧**:
1. 政策发布前 1-2 周开始有「政策预期」上涨, 此时可分批建仓
2. 政策正式发布当天往往是**短期顶**, 应减仓而非加仓 (典型「利好出尽」)
3. 如果政策超预期 + 行业本身有产业逻辑, 可保留 30-50% 仓位继续持有

### 第二类 业绩驱动 (季度性)

举例:
- 业绩预告期 (1 月、4 月、7 月、10 月) → 提前布局景气度高、订单充足的板块
- 高送转预案预期 → 提前布局每股资本公积 / 留存收益高、股价高的小盘股

**关键技巧**:
1. **业绩预增公告日往往是兑现日, 不是起点**——除非业绩远超市场一致预期
2. 关注**机构持仓变化** (季报披露) + **预期上修方向** (券商连续上修目标价)
3. 警惕**业绩预增 + 限售解禁同期**, 多半是出货行情

### 第三类 重组并购驱动

举例:
- 央企混改预期 / 重大资产重组停牌 / 国企整合 / 借壳上市

**关键技巧**:
1. **复牌当日定生死**: 大涨 = 利好兑现 (减仓), 大跌 = 重组失败 (清仓)
2. 并购预案公布后**等审批节奏**: 证监会受理、上会、批复, 每个节点都有一波
3. **并购失败案例**远比成功多, 不要重仓单一并购预期

### 第四类 生命周期事件

举例:
- **解禁**: 大股东 / 战投限售股到期, 通常前 1-2 周开始抛压, 利空
- **高送转**: 10 送 30、转增 + 派息, 短期股价催化, 但配股后填权与否要看基本面
- **大额分红**: 高股息标的的稳定现金流催化, 长期资金布局点

### 第五类 产业事件

举例:
- 苹果发布会前 1-2 月, 苹果产业链股票通常有催化
- 新能源车销量公告月报, 头部 EV 销量超预期 → 产业链拉升
- 国际地缘事件 (战争 / 制裁) → 军工 / 资源 / 能源 / 替代国产化

## 五、核心卖出逻辑

| 信号 | 处理 |
|---|---|
| 事件正式落地, 利好出尽 | 立即减仓 50%, 观察市场反应 |
| 反应不及预期 (期待暴涨, 结果平淡) | 全部清仓 |
| 反应超预期 (情绪共振, 量能放大) | 保留 30-50% 跟主升 |
| 出现量价背离 / 长上影 / 利好兑现新高滞涨 | 减仓 |
| 新事件冲击原事件 (新政策替代旧政策) | 立即清仓 |
| 持仓股突然停牌 / 业绩爆雷 | 复牌后第一时间走 |

> 事件驱动的核心心法: **永远在「确认 + 反应」之前撤退, 不要等市场告诉你「事件已经被消化了」**。

## 八、失效场景

1. **市场对事件免疫**: 政策反复出台但每次反应越来越弱, 此时该撤离
2. **黑天鹅事件冲击**: 突发的国际事件 / 监管利空, 让原本的事件预期完全失效
3. **冰点期**: 市场极度悲观, 任何利好都不反应, 任何利空都放大
4. **频繁切换事件**: 不要见一个事件追一个, 要选自己有研究、有判断的领域

## 九、常见误区

| 误区 | 真相 |
|---|---|
| 「政策出台就买」 | 错。政策正式出台往往是顶, 应该是「预期升温阶段」买 |
| 「业绩预增就买」 | 错。如果一致预期已经反映了 30% 的增长, 实际增长 30% 反而是利空 |
| 「重组停牌就赚」 | 错。重组失败概率 > 50%, 重仓单一重组等于赌 |
| 「高送转就涨」 | 部分正确。高送转有短期催化, 但填权与否看公司基本面, 别指望长期 |
| 「事件结束就回归基本面」 | 错。事件可能改变行业基本面 (如政策放松地产), 但绝不能"先涨后回归" |

## 十、和其他体系的关系

- 与**短线情绪体系**有 30% 重叠: 短线题材常因事件触发, 但短线只做发酵后 2-5 天, 事件驱动会更早布局、持有更长
- 与**宏观周期体系**强互补: 宏观决定大方向 (该不该重仓权益), 事件驱动在大方向里抓具体催化
- 与**中长线趋势体系**互补: 重大事件可能成为趋势的起点, 但中长线必须等趋势确立, 事件驱动可以提前布局
- 与**价值长线体系**互斥: 事件驱动周期太短, 与价值长线的"持有 5 年+"完全两套思路
