## 核心命题

> 没有"最好"的估值指标,只有"对的场景"。
> PE 适合稳定盈利的成熟公司,PB 适合周期股和金融股,PS 适合未盈利或转型期公司。
> **用错了估值指标,再精确的数字也是错的。**

## 一、问题的起源 —— 为什么"PE 8 倍很便宜"经常是陷阱

A 股散户最常见的估值困惑:"我看 PE 才 8 倍,很便宜啊,买了为什么还跌?"

这种困惑的根源,是把估值指标当成了**绝对真理**——以为 PE 低就是便宜,PE 高就是贵。但实际上:

- PE 8 倍对**消费股**是低估(消费股稳定盈利,8 倍意味着 8 年回本,远低于历史均值)
- PE 8 倍对**周期股**可能是**高估**(周期顶部 EPS 虚高,看起来 PE 低,其实是顶)
- PE 8 倍对**银行股**是合理(银行长期 PE 中枢就在 5-10 倍)

同一个 PE 8 倍,对三个行业的含义完全不同——这就是估值指标的"场景依赖性"。

更深一层:**每个估值指标都有其底层假设**。背离这个假设硬用,数字算得再准也是错的。这是一篇讲"什么时候用什么指标"的文章——不是讲怎么算估值,是讲怎么**选**估值方法。

## 二、底层原理 —— 估值指标的本质是什么

估值的终极理论是 DCF(未来现金流折现),但 DCF 太难用,所以人们发明了各种"代理指标":

- **PE**(市盈率)= 股价 / 每股盈利 = 市值 / 净利润
- **PB**(市净率)= 股价 / 每股净资产 = 市值 / 净资产
- **PS**(市销率)= 股价 / 每股营收 = 市值 / 营业收入

每个指标都是用一个"可观察的会计数据",代理"未来现金流的折现值"。它们各自的隐含假设不同:

| 指标 | 隐含假设 | 失效场景 |
|---|---|---|
| PE | 未来盈利稳定可预测 | 周期股(盈利大幅波动) / 早期成长股(盈利尚未释放) |
| PB | 净资产是公司清算 / 重置价值的合理代理 | 轻资产公司(科技、消费品牌)、净资产已被掏空的公司 |
| PS | 营收质量稳定可比,营收最终会转化为利润 | 不同毛利率的行业之间(电商 5% 毛利 vs SaaS 80% 毛利) |

理解了"假设",就能理解"为什么用错指标会出问题"。这是这篇文章的逻辑基础。

## 三、三个指标的适用矩阵

按公司类型,这是一张"该用什么指标"的速查表:

| 公司类型 | 首选指标 | 第二备选 | 理由 |
|---|---|---|---|
| 消费品龙头 | PE | PEG | 盈利稳定,PE 是最直接的回本时间 |
| 公用事业 | PE / 股息率 | EV/EBITDA | 盈利稳定,分红稳定 |
| 银行 | PB | 股息率 | 净资产 = 资本,PB 反映 ROE 与资本充足 |
| 保险 | PEV(内含价值) | PB | 利润延后释放,PEV 更准 |
| 券商 | PB | PE | 强周期,PB 比 PE 稳定 |
| 钢铁 / 煤炭 / 化工 | PB | EV/EBITDA | 强周期,盈利波动大,PE 失真 |
| 地产 | PB / NAV | 销售额对比 | 资产为主 |
| 互联网 / SaaS | PS | EV/Revenue | 成长期可能未盈利 |
| 创新药 | 管线估值 | PS | 盈利尚未释放 |
| 重资产周期(航空、海运) | EV/EBITDA | PB | 折旧大,PE 失真 |
| 科技硬件(消费电子) | PE | PS | 中度周期,PE 配合产品周期看 |

**核心原则**:**先判断公司类型,再选指标**。永远不要拿一个指标走天下。

## 四、推导:为什么不能用单一指标走天下

任何单一指标都隐含一个假设。这个假设在某些行业成立,在另一些行业完全错误。

**PE 的隐含假设是"盈利稳定"**——
- 消费品(茅台、伊利):假设成立 → PE 有效
- 周期股(钢铁、煤炭):假设不成立 → 周期顶部 EPS 是峰值利润,PE = 价格 / 峰值利润 = 看起来低,实则是顶 → **PE 越低反而越危险**

**PB 的隐含假设是"净资产 = 真实价值"**——
- 银行(资产就是钱):假设成立 → PB 有效
- 茅台(品牌价值远超账面净资产):假设不成立 → 茅台 PB 12 倍看起来贵,但你买的是品牌权,不是净资产 → **看 PB 等于忽略了茅台 99% 的价值**

**PS 的隐含假设是"营收质量稳定"**——
- 互联网平台(高毛利、高复购):假设成立 → PS 有效
- 不同行业之间:假设不成立 → 电商 PS 1 倍 vs SaaS PS 10 倍,不是 SaaS 高估,是行业本质不同 → **跨行业用 PS 比较毫无意义**

所以:**正确做法是先选对指标,再算出数字,然后和这只票自己的历史比较**。永远不要拿一只股票的 PE 去比另一只不同行业股票的 PE,这是估值最常见的错误。

## 五、各指标的具体使用方法

### 5.1 PE 的正确用法

**适用前提**:盈利稳定可预测,通常是消费、医药、公用事业等成熟行业。

**核心原则**:
- **比自己,不比别人**(不同行业不可比,同一只票看历史 PE-Band)
- **看 PE 在历史 5 年的什么分位**:< 30% 分位偏低估,> 70% 分位偏高估
- **配合 PEG 看**(PEG = PE / 未来 3 年增长率,< 1 低估,> 1.5 高估)
- **警惕"周期股 PE 陷阱"**:周期顶部 EPS 高,PE 看起来低实则是顶

**实操对照**(各行业 PE 中枢参考):

| 行业 | 历史合理 PE 中枢 |
|---|---|
| 白酒 | 25-35 倍 |
| 中药 | 20-30 倍 |
| 化学制药 | 25-40 倍 |
| 公用事业 | 12-18 倍 |
| 电力 | 15-20 倍 |
| 食品饮料 | 25-35 倍 |
| 银行 | 5-8 倍(注意是 PE) |

### 5.2 PB 的正确用法

**适用前提**:公司主要价值在净资产里(金融、地产、周期、重资产)。

**核心原则**:
- **看 PB-Band 历史分位**
- **必须配合 ROE 看**:高 ROE + 低 PB = 真便宜;低 ROE + 低 PB = 价值陷阱
- **"破净"不一定是机会**:很多破净公司是因为资产质量差、有大量减值风险
- **看"市净率隐含的 ROE"**:理论上 ROE 高的公司应该 PB 高,如果 ROE 高但 PB 低,可能存在错杀;反之如果 PB 高但 ROE 低,可能存在高估

**实操对照**:

| 行业 | 历史合理 PB 中枢 |
|---|---|
| 银行 | 0.6-1.2 倍 |
| 保险 | 1.0-2.0 倍 |
| 券商 | 1.2-2.0 倍 |
| 房地产 | 0.5-1.5 倍 |
| 钢铁 / 煤炭 | 0.5-1.2 倍 |
| 化工 | 1.0-2.0 倍 |

### 5.3 PS 的正确用法

**适用前提**:公司未盈利或盈利不稳定,但营收能反映规模和市场地位。

**核心原则**:
- **同行业可比,跨行业不可比**
- **必须配合营收增速看**:PS 5 倍 + 增速 50% = 合理;PS 5 倍 + 增速 5% = 高估
- **必须配合毛利率看**:80% 毛利的 SaaS 应该 PS 10 倍,5% 毛利的电商应该 PS 1 倍
- **警惕"刷营收"的公司**(应收账款飙升、关联交易暴增、毛利率持续下滑)

## 六、操作清单

- [ ] 第一步:**先判断公司类型**(消费 / 周期 / 金融 / 成长 / 重资产)
- [ ] 第二步:**选对指标**(消费用 PE / 周期用 PB / 成长用 PS / 重资产用 EV)
- [ ] 第三步:**看历史分位**(不要用绝对数字,用这只票自己的 5-10 年历史)
- [ ] 第四步:**配合质量指标用**(PE+PEG / PB+ROE / PS+毛利率)
- [ ] 第五步:**留 30% 安全边际**(估值是模糊的科学,精确的偏差比模糊的正确更糟)

## 七、三个最致命的估值陷阱

**陷阱一:周期股的 PE 陷阱**

2021 年钢铁、煤炭、化工 PE 跌到 5 倍以下,看起来"超低估",散户疯狂涌入。
但实际上:周期顶部 EPS 是峰值利润,**PE = 价格 / 峰值利润 = 看起来低,实则是顶**。
2022-2023 年盈利回归,EPS 跌 70%,PE 反而上升到 15-20 倍——**股价腰斩,但 PE "变贵"了**。这就是周期股 PE 陷阱。

**正确做法**:周期股看 PB(净资产相对稳定),配合行业景气度判断周期位置。

**陷阱二:消费股的 PB 陷阱**

有人说"茅台 PB 12 倍,太贵了"——这是估值方法用错了。
茅台的真实价值在品牌、渠道、定价权,不在账面净资产。**用 PB 看消费品牌,等于用秤量房子大小**——指标和对象不匹配。

**正确做法**:消费股看 PE-Band 和 PEG,不看 PB。

**陷阱三:成长股的 PE 陷阱**

2021 年宁德时代 PE 200 倍,按"绝对 PE 标准"是高估。
但当时增速 100%+ → PEG = 2,按 PEG 标准是合理偏高。
2022-2024 年增速回落到 20%,PEG 回到 5 以上 → 真正的高估出现。
**所以成长股不能只看 PE,必须配合增长率看 PEG**。否则你会在合理增速期错过,在增速回落期套牢。

## 八、A 股本土化注解

A 股估值有几个独特的难点,海外教科书没讲过:

**1. 行业 PE 标准会"漂移"**
A 股不同时期对同一行业的"合理 PE"标准会发生历史性漂移:
- 银行:2007 年合理 PE 30 倍,2024 年合理 PE 5-7 倍 → 漂移幅度极大
- 白酒:2010 年合理 PE 15 倍,2020 年合理 PE 50 倍 → 漂移幅度极大
- 地产:2015 年合理 PB 2 倍,2024 年合理 PB 0.4 倍 → 漂移幅度极大

应对:**用近 5 年滚动均值,不要用 10 年以上历史数据**——长期数据已经包含了过时的估值标准,会误导你。

**2. 国资股的"折价"和"溢价"**
央企国资股长期 PB 折价(市场担心治理风险、现金流不归股东),近年"中特估"行情后部分国资 PB 回升。
应对:国资股用"PB + 股息率"组合判断,单看 PB 容易误判——很多国资股 PB 0.5 倍但股息率 7%+,实际是隐藏的好资产。

**3. 主题股期估值指标全失效**
A 股的主题股(AI、机器人、低空经济、量子)经常炒"想象空间",PS 可以飙到 30-50 倍,PE 几百倍甚至无 PE(亏损)。
此时所有估值指标都失效,股价完全由情绪和资金面驱动。
应对:**主题股期不要用估值方法,改用情绪周期方法**(见短线类文档)。强行用估值买卖,在主升浪期会一直觉得"贵"而踏空,在退潮期会一直觉得"便宜"而套牢。

**4. A 股具体案例对照**

- **贵州茅台**:消费 → 用 PE-Band,历史中枢 30-35 倍 → 当前 PE > 50 偏高,< 25 偏低
- **招商银行**:金融 → 用 PB(0.8-1.5 倍中枢)+ 股息率(4%+ 偏低估)
- **宝钢股份**:周期 → 用 PB(0.6-1 倍中枢)+ 行业景气度(钢价 / 螺纹钢期货)
- **比亚迪**:成长 → 用 PEG,PE 单独看意义有限
- **中国神华**:周期 + 高分红 → 看股息率为主,> 7% 是机会区间
- **某 AI 主题股**:估值毫无意义,看资金面和情绪指标

**5. A 股的"假估值便宜"陷阱**
A 股有一类公司,各种估值指标都低,但常年不涨——这是"价值陷阱"。
特征:行业在萎缩 / 治理有问题 / 现金流不归股东。
识别:**便宜不是理由,要看为什么便宜以及便宜会不会被纠正**。如果"便宜"的原因是结构性的(行业萎缩、治理差),那便宜可能持续 10 年。

## 九、一句话记忆

> 选对指标比算准数字重要 100 倍。
> **用 PE 看周期股、用 PB 看消费股、用 PS 跨行业比较——这三个都是必亏操作。**
