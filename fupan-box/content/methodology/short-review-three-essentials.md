## 核心命题

> **不复盘的交易者,永远不会进步**。
> 顶级游资每晚都花 1-2 小时复盘——这才是他们和散户的真正差距。
> 复盘的核心三要素:**涨停分析(谁涨停 + 为什么)、龙头梳理(各板块龙头 + 资金动向)、明日预演(明日机会 + 风险点)**。
> 不是看了就算"复盘",**有产出(笔记 + 计划)的才是真复盘**。

## 一、问题的起源 —— 为什么"看了一晚上"还是不进步

很多散户晚上也"复盘"——刷股吧、看大 V 文章、看新闻、看 K 线。
但**第二天还是抓不住机会,还是被套**。为什么?

因为这种"复盘"本质是**信息消费**,不是**主动思考**。

真正的复盘有三个特征:
1. **有结构**(按方法论一步一步做,不是随便看)
2. **有产出**(写下来 + 形成明日计划,不是"在脑子里想想")
3. **有验证**(明日操作的结果反过来验证今晚的判断,不断迭代)

顶级游资的复盘是**仪式化、流程化**的——每天 1-2 小时,雷打不动。
而散户的"复盘"是**碎片化、消费化**的——刷刷帖子,看看视频,然后睡觉。

差距就在这里。

## 二、底层原理 —— 为什么复盘是短线最大的"复利源"

复盘的本质是"**用昨天的经验,改进明天的决策**"。

数学上:
- 没有复盘:每天都重复相同的错误 → 长期看,**胜率不变**
- 有复盘:每天发现 1-2 个错误并改正 → **胜率每月提升 1-2%**
- 1 年下来,**胜率从 40% 提升到 60% +**(质变)

更重要的是,复盘形成**"经验闭环"**:
```
观察(白天交易) → 反思(晚上复盘) → 改进(明日操作)
                ↑                              ↓
                └──── 新经验,新观察 ←─────┘
```

每一次循环,你的判断力都在提升——这就是"**经验复利**"。

不复盘的人,1 年的经验**重复了 365 次**;复盘的人,1 年是**365 次不同的经验积累**。差距是指数级的。

## 三、复盘三要素详解

### 要素 1:涨停分析 —— "今天谁涨停 + 为什么"

**目的**:**理解今日资金的偏好**

**具体步骤**:

**第一步:统计涨停数据**
- 今日涨停总数(多少只)
- 今日跌停总数(多少只)
- 涨停 / 跌停 比例(健康度判断)

**第二步:涨停板分类**
按涨停原因分类,理解"今日资金偏好":
- **板块龙头**(明确的板块主线)
- **题材股**(政策 / 事件驱动)
- **业绩超预期**(基本面驱动)
- **重组 / 并购**(消息驱动)
- **跟风涨停**(板块内跟风)
- **末位补涨**(退潮信号)

**第三步:重点票分析**
对每只**重要涨停**(高度连板 / 板块龙头 / 异动)详细分析:
- 涨停时间(早盘 / 上午 / 尾盘)
- 涨停原因(消息 / 技术 / 跟风)
- 量能 + 换手
- 龙虎榜资金(谁买谁卖)
- 是否有"分歧 / 一致" 信号

**第四步:形成涨停日志**
建立 Excel / Notion / 笔记本,记录每天:
- 高度龙头(几板)
- 板块龙头
- 异动票
- 龙虎榜资金动向

**1 个月后回顾**,你会发现:
- 哪些板块持续活跃
- 哪些游资在持续运作
- 哪些涨停模式稳定有效

### 要素 2:龙头梳理 —— "各板块的龙头 + 资金动向"

**目的**:**找到明日的"接力候选"**

**具体步骤**:

**第一步:列出今日活跃板块**
- 涨停数 ≥ 3 只的板块
- 板块当日涨幅 ≥ 3%
- 关注资金净流入板块

**第二步:每个板块识别龙头**
按以下标准:
- **高度龙头**:板数最高的票
- **空间龙头**:涨幅最大的票(从底部算起)
- **量能龙头**:成交量最大的票
- **认知龙头**:市场公认的板块代表

**第三步:龙头资金分析**
对每个龙头看龙虎榜:
- 买方席位:**顶级游资 / 机构 / 量化**
- 卖方席位:**部分获利出场 vs 主力派发**
- 净买入金额

**第四步:判断龙头健康度**
- 唯一性:是否有竞争对手(唯一性强 = 健康)
- 量能配合:成交是否合理
- 龙虎榜:**顶级游资买入 = 健康;游资卖出 = 警觉**

**第五步:形成龙头候选**
明日重点关注:
- **今日新晋龙头**(首板异动,可能成为新龙头)
- **持续龙头**(已经几板,接力候选)
- **板块二线**(龙头分歧后的接力候选)

### 要素 3:明日预演 —— "明日的机会 + 风险点"

**目的**:**形成明日操作计划**

**具体步骤**:

**第一步:重大事件预判**
- 明日有什么重要数据 / 政策预期?(经济数据 / 政策吹风)
- 明日有什么公司公告?(财报 / 重组 / 业绩预告)
- 明日有什么外部影响?(美股表现 / 港股表现 / 国际事件)

**第二步:技术形态判断**
- 大盘明日大概率方向(高开 / 低开 / 平开)
- 重点票明日的关键位置(支撑 / 阻力)

**第三步:列出明日**"重点池"
按优先级:
- **A 池**(必须关注):高度龙头、板块龙头
- **B 池**(机会关注):新晋异动、龙头候选
- **C 池**(背景关注):大盘走势、板块情绪

**第四步:形成明日操作计划**
具体到:
- **买什么**(候选股 + 买入条件 + 买入价位)
- **卖什么**(持仓股的止损位 / 减仓条件)
- **仓位计划**(总仓位 + 单股仓位)
- **风险预案**(如果大盘急跌 / 急涨怎么办)

**第五步:记录明日预期**
写下你对明日的判断:
- 大盘方向
- 板块强弱排序
- 重点票走势预期

**第二天盘后对照**,看你的判断有多准——这是**判断力训练的最佳方法**。

## 四、复盘的标准时间表

```
15:00-15:30 当日复盘(简版)
- 总结当日操作
- 标记当日错误
- 记录情绪状态

15:30-17:00 自由时间(休息 / 散步)

19:00-21:00 深度复盘(完整三要素)
- 19:00-19:30 涨停分析
- 19:30-20:30 龙头梳理 + 龙虎榜
- 20:30-21:00 明日预演 + 操作计划

21:00-22:00 学习时间
- 看研报 / 看大 V
- 学新方法 / 看历史案例

22:00 睡觉
```

## 五、推导:为什么"碎片化复盘"无效

很多人晚上也花了 1-2 小时,但效果很差。原因:

**问题 1:无结构**
只是"刷帖子",没有按方法论逐步做。
导致信息进入大脑但**没有形成认知**。

**问题 2:无产出**
看了不写。
**没有书写,就没有思考**——书写是把"模糊的感觉"变成"清晰的判断"的工具。

**问题 3:无对照**
今晚的判断,明天没有对照验证。
导致"判断永远停留在主观,无法优化"。

**修正方法**:
- 用**模板化笔记**(强制按结构记)
- **每天写复盘日志**(强制有产出)
- **第二天对照验证**(强制有反馈)

## 七、操作清单

### 每日盘后必做(15:00-15:30)

- [ ] 总结当日操作(买卖记录 + 盈亏)
- [ ] 标记当日的"错误操作"(为什么错)
- [ ] 记录情绪状态(贪婪 / 恐惧 / 冲动)

### 每晚深度复盘(19:00-21:00)

- [ ] **涨停分析**:统计、分类、重点票分析
- [ ] **龙头梳理**:列板块、找龙头、看龙虎榜
- [ ] **明日预演**:事件预判、技术判断、操作计划

### 每周日总复盘

- [ ] 本周操作的胜率 / 盈亏统计
- [ ] 本周的"最大错误"和"最大成功"
- [ ] 下周的策略调整

### 每月底总复盘

- [ ] 本月盈亏 / 胜率
- [ ] 行情判断准确率
- [ ] 策略适用性
- [ ] 下月调整方向

## 八、三个常见误区

**误区一:"看了一晚上就算复盘"**
看 ≠ 思考,看 ≠ 复盘。
**修正方法**:**必须有结构 + 有产出**(写下来,不要只在脑子里想)。

**误区二:"复盘只看自己持仓的票"**
只看自己的 → 视野窄 → 错过更大机会。
**修正方法**:**复盘要看全市场涨停板 + 龙头**,不只是自己的持仓。

**误区三:"复盘只看技术,不看资金"**
只看 K 线,不看龙虎榜 / 资金流向 → **判断会片面**。
**修正方法**:**技术 + 资金 + 消息三位一体地看**。

## 九、适用与不适用

- ✅ **适用**:所有 A 股短线 / 波段交易者
- ✅ **适用**:打板族、接力族、主题股玩家
- ⚠️ **半适用**:中长线价值投资(复盘频率可以低,但财报季要复盘)
- ❌ **不适用**:量化高频(机械化运行,不需要主观复盘)

## 十、A 股本土化注解

**1. A 股短线节奏快,复盘的及时性极重要**
A 股短线机会"稍纵即逝"——主升浪 5-10 天就结束。
应对:**复盘必须每日做,不能"周末再补"**。延迟一天,就错过最佳节奏。

**2. 信息渠道的差异**
A 股有大量"独家信息源",顶级游资有自己的信息圈。
散户的劣势:信息慢一拍。
应对:**关注顶级游资的复盘文 / 微博 / 公号**(免费的也有不少,要筛选优质来源)。

**3. 政策市的复盘重点**
A 股是政策市,**政策动向是复盘的重要内容**。
应对:**每晚关注国务院 / 央行 / 证监会等的最新动向,可能改变明日行情**。

**4. 复盘日志的价值**
A 股变化快,**3-6 个月前的复盘日志,反过来看会让你学到很多**。
应对:**坚持写复盘日志,定期翻看历史**(每季度回顾一次)。

**5. A 股具体复盘案例**

- **2023 年 ChatGPT 主题**:坚持复盘的人,提前发现"AI 主题逐步发酵" → 在主升期前进场
- **2024.9 政策市**:坚持复盘的人,在 9.24 当天就发现"政策开始转向" → 第二天重仓
- **2024.10 退潮**:坚持复盘的人,在退潮初期就发现"涨停数从 100+ 降到 50" → 第一时间减仓
- **某主题股退潮**:坚持复盘的人,看到"高度龙头炸板 + 末位补涨" → 立刻清仓避免大亏

**6. 不同人群的复盘强度**

| 人群 | 复盘强度 | 时间投入 |
|---|---|---|
| 顶级游资 | 极强(每日 2-3 小时) | 每周 15-20 小时 |
| 中级短线 | 强(每日 1-2 小时) | 每周 7-14 小时 |
| 业余短线 | 中(每日 30-60 分钟) | 每周 3-7 小时 |
| 中长线持仓 | 弱(每周 2-3 小时) | 周末集中复盘 |
| 价值投资 | 极弱(每月几次) | 财报季集中复盘 |
