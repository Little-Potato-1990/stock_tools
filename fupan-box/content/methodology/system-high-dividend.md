## 核心命题

> 高股息体系是「**把股票当类债持有, 用稳定股息率作底, 用估值修复作弹性**」。
> 核心收益来源 = 股息率 + 估值修复 + 适度成长, 三者共振时年化可达 15-25%, 静态时也有 5-8% 的稳定股息。
> 这是 A 股**最适合防御 + 长期复利**的体系, 也是**最不依赖择时**的体系。

## 一、这个体系到底在做什么

高股息体系的世界观非常简单:

**一只股票 = 一个永续债 + 一个看涨期权**。
- 永续债部分 = 每年的股息, 越稳定越好
- 看涨期权部分 = 估值修复 + 业绩增长

如果一家公司能稳定每年分红 5%、且公司不死, 那么:
- 持有 20 年, 光股息就能回本
- 期间股价波动多大, 都不影响你拿到股息
- 利率下行时, 5% 股息率显得越来越香, 估值会被买上去
- 公司本身如果还有 5-10% 的业绩增长, 那就是双击

> 高股息的精髓是「**让企业每年用真金白银的现金流, 来证明自己**」。
> 这比任何故事、任何 PPT、任何 K 线信号都可靠。

**A 股 2023-2024 年「中特估行情」**就是高股息体系的一次集中演绎:
- 中国神华 / 长江电力 / 工商银行 等长期被冷落的高股息央国企, 估值修复 + 股息再投, 年化收益 30%+
- 这些股票在牛市里涨得慢、在熊市里跌得少、在震荡市里靠股息吃饭

## 二、目标周期与仓位结构

- **持有周期**: 1-5 年, 极少卖出
- **仓位构成**: 8-15 只行业分散的高股息股, 单股不超过 15%, 现金 5-15%
- **目标年化**: 8-15% (基础) / 15-25% (估值修复年份)
- **最大回撤接受度**: -10% 到 -15% (远低于其他权益体系)
- **关键指标**: 组合的**加权股息率**, 应稳定在 4.5% 以上

## 三、适合什么人

适合:

- 退休账户、家庭闲钱、对回撤敏感
- 不愿天天看盘, 季度调整一次就够
- 接受「年化 10% 不刺激但稳」, 而非「年化 50% 但回撤 30%」
- 资金量较大 (50 万以上, 才能体现现金流稳定性)
- 长期通缩 / 利率下行预期下的核心资产配置

不适合:

- 想短期翻倍的人
- 资金量太小 (5 万以下, 股息再投复利效应不明显)
- 高利率环境下还重仓高股息 (这时债券收益率更高)
- 完全不关心估值、纯吃股息的人 (估值高位时高股息也会大跌)

## 四、核心买入逻辑 (4 道闸门)

### 第一道:股息率 ≥ 历史均值的 1.2 倍

不是单纯「股息率高」就买, 而是看**当前股息率相对自身历史的位置**。

举例:
- 长江电力历史平均股息率 3-4%, 当前股息率 5%+ → **可买**
- 中国海油历史平均股息率 5-6%, 当前股息率 8%+ → **强烈可买**
- 某只股票股息率 10%, 但因为股价腰斩 → **要看公司基本面, 可能是陷阱**

### 第二道:股息可持续性 (3 个核心条件)

参见《[ROE 与杜邦分析](value-roe-dupont)》。

1. **分红比例稳定**: 过去 5 年现金分红率稳定在 30-70%
2. **现金流支撑**: 经营性现金流 / 净利润 > 80% (净利润不能"水分太大")
3. **资产负债结构稳健**: 资产负债率 < 60% (不靠借钱分红)

### 第三道:行业属性 (有特征性偏好)

历史上长期跑赢的高股息行业:

| 行业 | 股息特征 | 风险 |
|---|---|---|
| **银行** | 股息率 4-7%, 估值低 | 房地产风险传染 |
| **公用事业** (电力 / 高速 / 水务) | 股息率 4-6%, 极稳 | 政府限价 |
| **能源** (煤炭 / 油气) | 股息率 6-10%, 有周期 | 大宗价格波动 |
| **运营商** (中移动 / 中电信) | 股息率 4-6%, 现金牛 | 增长停滞 |
| **REITs / 基础设施** | 股息率 5-8% | 利率敏感 |
| **核心消费** (白酒龙头 / 必需消费) | 股息率 2-4%, 但成长性好 | 估值偏高 |

### 第四道:估值不离谱

参见《[PE/PB/PS 什么时候用](value-pe-pb-ps-when-to-use)》。

- PE 不在历史 80 分位以上
- PB 不显著高于历史中位数
- 股息率不在历史 30 分位以下 (股息率太低 = 股价太高)

## 五、核心卖出逻辑 (有但很少)

> 高股息体系**最大的优势是不卖**。频繁切换会损失复利效应。

只有以下情况才卖:

| 信号 | 处理 |
|---|---|
| 公司宣布大幅减少分红 (如分红率从 50% 降到 20%) | **立即清仓**, 这是最严重信号 |
| 经营现金流连续两年负 | 清仓 (无法支撑分红) |
| 股息率跌破历史 30 分位 (股价大涨稀释收益率) | 减仓 50% 锁利润 |
| PE / PB 进入历史 90 分位以上 | 减仓 30% (估值泡沫) |
| 行业进入永久性衰退 (如传统电信被替代) | 逐步换仓到新的高股息 |

## 八、失效场景

1. **高利率环境**: 当 10 年期国债收益率超过 6%, 股息率 5% 的高股息股就没有吸引力
2. **行业被颠覆**: 如传统燃油车被电车替代、传统电视被流媒体替代——产业级风险
3. **极端通胀**: 高通胀下, 名义股息率被实际购买力侵蚀
4. **疯牛末期**: 题材股鸡犬升天, 高股息股几个月不动, 但通常 3-6 个月后回归
5. **公司治理风险**: 央企突然大幅资本开支扩张、停止分红, 个股黑天鹅

## 九、常见误区

| 误区 | 真相 |
|---|---|
| 「股息率越高越好」 | 错。股息率 > 10% 往往是股价腰斩造成的「假象高股息」, 要先看公司是否健康 |
| 「分红除权后股价跌了, 没赚到」 | 错。分红是真金白银到手, 除权是会计处理, 长期看会填权 |
| 「高股息没成长, 不值得拿」 | 半对。高股息 + 适度成长才是最佳, 但纯防御策略也有价值 |
| 「央国企永远不会破产」 | 错。央国企有政策保护, 但也有国资改制 / 退市可能, 不要把单一公司当无风险 |
| 「我现在用不上股息, 不用买」 | 错。股息再投资才是高股息体系的核心, 长期复利极其惊人 |

## 十、和其他体系的关系

- 与**价值长线体系**强重叠: 高股息可看作价值长线的一个特化分支, 都重视基本面 + 长期持有
- 与**中长线趋势体系**互补: 高股息可作为「防御基金」, 中长线趋势作为「进攻基金」
- 与**宏观周期体系**强相关: 宏观决定何时该重仓高股息 (利率下行 / 通缩 / 衰退象限)
- 与**短线情绪体系**完全互斥: 不要把高股息股做成短线题材
- 与**事件驱动体系**互斥: 高股息追求稳定, 事件驱动追求催化, 两套逻辑
