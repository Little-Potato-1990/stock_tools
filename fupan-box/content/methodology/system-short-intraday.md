## 核心命题

> 短线情绪不是「赌涨停」, 是「**在合适的市场氛围下, 站在最有性价比的梯队位置, 等下一棒接力盘**」。
> 你赚的不是公司基本面, 也不是估值修复——你赚的是**情绪从冷到热再到冷的资金切换差**。
> 这套体系**赢面其实不大, 但只要严守纪律, 赔率可以做得很高**。

## 一、这个体系到底在做什么

A 股有一个海外市场不存在的现象:**T+1 + 涨跌停板 + 散户为主**, 三者叠加, 制造出全世界独一无二的「**情绪驱动型短线生态**」:

- 涨停板是**情绪的可视化锚点**: 每天涨停的股票数量、连板数量、首板数量, 直接反映市场温度
- 题材接力是**资金的击鼓传花**: 同一个题材里, 龙头 → 二线 → 跟风 → 末位, 接力顺序极其规律
- 游资席位是**资金风格的指纹**: 同一个营业部反复操作类似题材, 形成可识别的「派系」

短线情绪体系的核心**不是选股**, 而是**判断当前市场处在情绪周期的哪一段**:

```
退潮期 → 冰点 → 启动 → 发酵 → 高潮 → 分化 → 退潮
   (空仓)  (试单)  (重仓)  (满仓接力)  (减仓)  (清仓)  (空仓)
```

> 短线交易的最大风险**不是选错股, 是选错时**。在退潮期做接力, 神仙战法都救不了你。

## 二、目标周期与仓位结构

- **持有周期**: 1-5 天为主 (隔夜 + 1-3 个交易日), 极少超过 1 周
- **仓位构成**: 同时持有 1-3 只, 单股可重仓 30-50%, 现金 20-100% 视情绪温度而定
- **目标年化**: 50-150% (高潮期) / -10% 到 30% (一般年份) / -30%+ (退潮期不停手)
- **最大回撤接受度**: 单年不超过 -25%, 单笔不超过 -8%
- **胜率/盈亏比目标**: 胜率 35-45%, 盈亏比 ≥ 3 (一次大利润抵 3-5 次小损)

## 三、适合什么人

适合:

- 能全天盯盘 (集合竞价 9:15-9:25 + 早盘 9:30-10:30 + 尾盘 14:30-15:00)
- 性格果断, 决策快, 错了能立刻砍
- 喜欢节奏快、反馈密集的交易
- 心理素质好, 涨停板封死和炸板都能保持冷静
- 愿意做大量复盘、跟踪数十个游资席位

不适合:

- 上班族 (这个体系**真的需要全天盯盘**, 不是看一眼)
- 资金量过大 (单票 500 万以上, 会面临流动性挑战)
- 输不起短期回撤的人
- 没纪律的人 (没纪律 = 注定爆仓)
- 把短线当「一次性赌博」的人 (长期负期望)

## 四、核心买入逻辑 (情绪周期 + 题材生命周期 + 卡位 三层过滤)

### 第一层 大势/情绪过滤 (非常重要)

**先判断今天能不能做, 再判断做什么**——参见《[市场环境过滤器](short-market-condition-filter)》。

简化版指标:

| 指标 | 安全区 | 危险区 |
|---|---|---|
| 涨停板数量 | > 60 | < 30 |
| 连板高度 | ≥ 5 板 | ≤ 3 板 |
| 涨停打板成功率 | > 70% | < 50% |
| 大盘日内走势 | 早盘红 / 强反 | 早盘绿 / 跳水 |

**只要任何一项进入危险区, 当日不开新仓**。

### 第二层 题材生命周期判断

每个题材都有一个生命周期, 短线只在特定阶段做——参见《[题材生命周期](short-theme-lifecycle)》:

```
潜伏期 (1-3 天) → 启动期 (1-2 天) → 主升期 (2-4 天) → 分歧期 (1-2 天) → 退潮期
   (难捕捉)         (首板战法)         (接力战法)         (减仓)        (空仓)
```

**主升期 + 早期分歧** 是短线最赚钱的时段, 退潮期是亏钱的时段。

### 第三层 个股卡位 (梯队战法)

参见《[涨停板梯队](short-limit-up-ladder-stages)》《[接力与卡位](short-relay-positioning)》。

每个题材都会形成「龙头-二线-跟风-末位」梯队, 不同位置打法不同:

| 卡位 | 风险 | 收益 | 适合时段 |
|---|---|---|---|
| 题材龙头 (高度板) | 高 | 高 | 主升期, 一字板战法 |
| 题材二线 (中度板) | 中 | 中 | 主升期, 接力战法 |
| 题材末位 (低度板) | 低 | 低 | 启动期, 卡位战法 |
| 补涨股 (未涨停) | 极低 | 极高/极低 | 分歧期, 低吸战法 |

### 第四层 龙虎榜验证

收盘后看龙虎榜, **看席位, 不看股票**——参见《[龙虎榜读法](short-lhb-decoding)》。

强势席位 (章盟主、赵老哥、东方财富拉萨等) 介入的题材龙头, 第二天接力概率高;
游资砸盘 (机构席位卖出) 的高位连板股, 第二天大概率开盘卖飞。

## 五、核心卖出逻辑 (秒级反应)

短线没有「中线持有」这个选项, 离场比入场更重要:

| 信号 | 处理 |
|---|---|
| 隔夜炸板 (开盘冲高未板回落) | 早盘第一时间走 |
| 题材一致性破坏 (其他票不跟) | 立即减仓一半 |
| 大盘跳水 (大跌 1%+) | 全部减仓不商量 |
| 个股长上影 / 天量大阳 | 收盘前必走 |
| 持仓股位列龙虎榜机构卖出席位 | 隔日开盘走 |

> 短线的本质就是**接力**——只要下一棒不存在, 你就必须当那个倒数第二棒, 不是最后一棒。

## 八、失效场景 (这些时候必须空仓)

1. **退潮期到冰点**: 涨停板数 < 30, 连板高度 ≤ 3 板, 任何题材都做不起来
2. **指数大跌日**: 大盘跌幅 > 1.5%, 所有打板都会高开低走
3. **政策利空发酵期**: 监管点名、IPO 暂停、印花税等突发事件
4. **个人状态差时**: 连续 3 笔亏损 / 情绪化 / 急于回本——立刻停手 3 天
5. **全市场风格切换日**: 比如「中特估单边吃肉」「白马反弹」这种日子, 题材股全军覆没

## 九、常见误区

| 误区 | 真相 |
|---|---|
| 「打板赚钱快」 | 长期看, 90% 短线散户亏钱。打板对纪律和盘感的要求, 远高于其他体系 |
| 「龙头股安全」 | 错。龙头股回撤更快, 一旦被砸是断头铡 |
| 「跟着游资买就行」 | 错。你看到龙虎榜时已是收盘后, 游资可能第二天就走 |
| 「拿过夜更稳」 | 错。短线最大的风险就是过夜, 隔夜利空 / 炸板让你毫无机会 |
| 「冰点期低吸」 | 极危险。冰点期不是底, 可能还有更冷的冰点 |
| 「我有手感」 | 错。盈利期的"手感"是市场给的, 退潮期会还回去 |

## 十、和其他体系的关系

- 与**价值长线体系**完全互斥, 资金账户必须严格分开
- 与**中长线趋势体系**互斥, 不要试图把"短线被套"变成"中线持有"
- 与**技术波段体系**有相通之处 (都看量价), 但短线更看消息和情绪, 周期短得多
- 与**事件驱动体系**有 30% 重叠 (题材爆发常因事件触发), 但短线只在事件发酵的 2-5 天里做
- 与**宏观周期体系**正交: 宏观决定大风格, 短线决定具体题材
