## 核心命题

> 龙虎榜是 A 股短线最重要的信息源,但 **90% 的散户读不懂**。
> 它不是"哪些股票上榜",而是"**哪些席位在买卖什么股票**"——后者才是真正的 alpha。
> **席位是资金的指纹,指纹会重复,所以可以跟踪。**

## 一、问题的起源 —— 一个意外催生 A 股游资文化的制度

龙虎榜是上交所 / 深交所每日公布的"异动股票交易席位明细"。最早从 2003 年开始公布,目的是"提高市场透明度,防范市场操纵"。

但意外的是——它催生了 A 股独有的"**游资文化**"。

为什么会这样?因为在中国 A 股,大资金交易必须通过券商席位,而席位是相对固定的(同一个游资团队多年使用同一个营业部席位)。**当这些席位反复在龙虎榜留痕时,游资风格逐渐形成可识别的"指纹"**:

- **章盟主**(常用席位:中信上海溧阳路营业部)→ 擅长高位接力大龙头
- **赵老哥**(常用席位:华泰上海雷峰塔)→ 擅长首板狙击 + 一日游
- **宁波桑田路**(华泰宁波解放南路)→ 擅长短线快进快出
- **佛山系**(主要在广发佛山等)→ 擅长低位埋伏 + 题材爆发
- **炒新一族**(华西成都人民南路等)→ 擅长新股次新股

跟踪这些席位,本质上是在跟踪 A 股最有 alpha 的一群短线资金。

但散户最常见的错误,是把龙虎榜当成"哪些股票成交活跃"的工具——看了半天,只记住了"哪些股票上榜",完全忽略了"谁在买谁在卖"。**这就丢失了龙虎榜 90% 的信息。**

## 二、底层原理 —— 为什么"看席位"比"看股票"有效

理解龙虎榜,必须先理解一个核心事实:

**个股是结果,席位是原因。**

- 一只股票为什么涨停?因为有资金推动
- 这些资金是谁?席位告诉你
- 这些资金的风格是什么?席位画像告诉你
- 这些资金明天会怎么做?**只要席位风格稳定,他们的行为就可预测**

这就是"跟席位"的根本逻辑——你不是在跟股票,而是在跟"特定资金的行为模式"。

由此推导出几个反常识的结论:

**1. 同一只股票,买方席位不同,意义完全不同**
- 章盟主净买入 5000 万 → 大概率次日继续推动
- 散户营业部净买入 5000 万 → 大概率是接力盘踩踏
- 机构席位净买入 5000 万 → 中线资金,与短线节奏不同步

**2. 同一个席位的反复出现,本身就是信号**
- 同一个游资 3 天连续上榜买入同一只股票 → 强信号
- 一个游资从没碰过的板块突然大额买入 → 弱信号(不是他的擅长领域)

**3. 顶级游资的"卖出"比他们的"买入"更值钱**
- 章盟主在某只票连续买了 3 天后突然出现在卖方 → 强烈警觉
- 这种"内部信号"是龙虎榜最有 alpha 的部分

理解了这一点,就理解了为什么"看席位 > 看股票"——**席位反映的是资金的'人',而'人'的行为模式比'股票'本身的随机走势更可预测**。

## 三、龙虎榜的四个核心读法

### 读法 1:买卖席位结构

任何一份龙虎榜,先看买方 5 个席位 + 卖方 5 个席位的"成分":

| 买方结构 | 卖方结构 | 信号判读 |
|---|---|---|
| 知名游资 + 散户营业部 | 散户营业部 | 短线积极信号(游资在拉,散户在跑) |
| 知名游资 + 知名游资 | 散户营业部 | 强信号(多游资共识) |
| 散户营业部 | 知名游资 | **危险信号**(游资出货,散户接盘) |
| 知名游资 | 知名游资 | 当日博弈激烈,胜负未定 |
| 一家独大(单席位 > 50%) | - | 单一游资坐庄,跟节奏 |
| 机构席位买入 | - | 中线资金进场,不同节奏 |
| - | 机构席位卖出 | 机构出货,中线警觉 |

**核心原则**:**看到顶级游资在买、散户在卖,跟随;看到散户在买、游资在卖,反向**。

### 读法 2:席位风格画像

要真正用好龙虎榜,必须建立 5-10 个核心游资席位的"风格画像档案"。下面是几个典型(具体席位代号会随时间变化,但风格相对稳定):

**章盟主(代表中信上海溧阳路)**
- 擅长:高位接力、龙头股
- 风格:重仓出击,持仓 1-3 天
- 信号意义:上榜买入 → 该票大概率有 1-2 天继续溢价
- 警觉:上榜卖出 → 高度龙头大概率见顶

**赵老哥(华泰上海雷峰塔)**
- 擅长:首板狙击、一日游
- 风格:经常打首板,次日高开就出
- 信号意义:上榜买入首板股 → 该题材次日高开概率高,但要快进快出
- 警觉:跟随他的票,**绝不能持有超过 2 天**

**宁波桑田路(华泰宁波解放南路)**
- 擅长:短线快进快出、波段
- 风格:多席位协同,分仓操作
- 信号意义:多席位同时买入 → 题材确认信号

**炒新一族(华西成都人民南路)**
- 擅长:新股、次新股
- 风格:专注新股次新股板块
- 信号意义:对老股票上榜意义不大,对新股上榜是强信号

**机构专用席位**
- 通常代表公募基金、保险、社保等
- 风格:中线持仓,不参与短线博弈
- 信号意义:**与游资节奏完全不同**,机构买入后的票不适合短线接力(机构持有久,短线无溢价)

(注:具体席位代号每年都会被市场更新和调整,以上风格归纳来自市场公开认知,实操中需要持续跟踪验证)

### 读法 3:净买入额对比

不要只看"上榜了",要看"买卖力量对比":

| 净买入 / 总成交占比 | 信号强度 |
|---|---|
| > 30% | 强信号(资金高度集中买入) |
| 15% - 30% | 中等信号 |
| < 10% | 弱信号(席位之间互砸,胜负未定) |
| 净卖出 > 总成交 20% | 出货信号 |

**单席位净买入超 5000 万**:接近游资单次出击的上限,代表强烈看好。
**多席位同时净买入**:题材确认,接力概率高。
**净买入但席位之间互砸**:博弈激烈,胜负不明,谨慎。

### 读法 4:连续上榜模式

观察一只股票连续几天的龙虎榜,可以判断主力意图:

| 模式 | 解读 | 操作信号 |
|---|---|---|
| 连续 3 天上榜买入 | 持续题材,资金共识强 | 中线接力机会 |
| 上榜买入次日"一日游"形态 | 游资撤退 | 立刻清仓 |
| 上榜买入后机构跟进 | 中线资金接力 | 持有可拉长 |
| 同一席位连续 3 天买入 | 单一游资坐庄,跟其节奏 | 看该游资风格决定持仓时长 |
| 顶级游资突然出现在卖方 | 主力出货信号 | 立刻清仓 |
| 多个不同游资轮番上榜 | 题材热度高,接力者多 | 板块行情确认 |

## 四、推导:为什么"跟席位"比"跟个股"有效

这是一个简单的概率推理:

1. 个股的明日走势 = 受多种力量影响(资金、消息、大盘、情绪)
2. 但其中"资金"是核心驱动 → 知道资金的行为模式可以预测一大半
3. 资金的行为模式 = 席位风格(因为同一席位往往是同一团队)
4. 席位风格相对稳定(团队的人不变) → 可跟踪可预测

所以**跟席位 = 跟一群可预测的资金**,胜率显著高于"看图猜走势"。

更深一层:**席位反映的是资金的"性格",而性格比"基本面"更稳定**。茅台的基本面 1 年内不会变化太多,但茅台的股价 1 个月可以波动 30%——后者就是资金性格的体现。把握资金性格,就是把握短期价格波动的关键。

## 五、操作清单

- [ ] 每日盘后必看龙虎榜(收盘后约 18:00 公布)
- [ ] 重点关注 5-10 个核心游资席位的当日动向
- [ ] 对照席位风格判断"明天会接力还是出货"
- [ ] 净买入 > 总成交 30% 的票重点关注
- [ ] 机构 + 游资同时买入 → 强信号
- [ ] 顶级游资突然出现在卖方 → 立刻清仓信号
- [ ] 散户席位主导买入 + 游资在卖 → 反向警觉
- [ ] 建立自己的"游资席位风格档案",每月更新

## 六、三个常见误区

**误区一:只看"上榜票"不看"席位"**
"今天 XX 股上榜了,我也买!" — 这是丢掉了 90% 的信息。
同一只票上榜,如果是顶级游资在买 vs 散户在买,意义完全相反。
**修正方法**:看到上榜票,第一动作不是看股票名字,是看买卖席位列表。

**误区二:盲目跟某个游资**
有人迷信"跟着赵老哥就能赚钱" — 但赵老哥是一日游风格,你跟了之后第二天必须卖,大多数散户做不到。
**修正方法**:跟席位之前,**必须先理解这个席位的风格**,然后用对应的持仓周期去跟。
- 跟一日游游资 → 必须 T+1 卖
- 跟接力游资 → 持仓 2-3 天
- 跟机构 → 持仓 1-3 个月

**误区三:忽略"反向用法"**
龙虎榜不只是"跟随信号",更是"反向警觉信号"。
- 顶级游资在某只票连续买 3 天后突然在卖方出现 → 该票大概率见顶
- 多个游资同时清仓某个题材 → 该题材热度结束
**修正方法**:对你已持仓的票,每天查龙虎榜,看自己跟随的席位是否在卖。

## 七、适用与不适用

- ✅ **适用**:短线打板、连板接力、题材股跟踪、热点板块识别
- ⚠️ **半适用**:中线持仓(机构席位有参考价值,但游资席位无关)
- ❌ **不适用**:长期价值投资(席位变动不影响公司基本面)
- ❌ **不适用**:大盘指数判断(龙虎榜只反映个股资金,不代表全市场)

## 八、A 股本土化注解

**1. 龙虎榜规则的几次关键变化**

- **2017 年**:对外公开了游资席位的具体营业部信息,游资识别成为可能
- **2020 年**:科创板 / 创业板纳入龙虎榜,但触发标准与主板不同
- **2024 年**:进一步细化披露,部分新增"机构专用席位"统计

应对:**关注规则变化,旧的席位画像可能因规则调整而失效**。

**2. 主要交易所的龙虎榜触发标准**(简化版)

| 类型 | 主板触发标准(示例) |
|---|---|
| 单日涨跌幅 | ≥ ±7%(主板,科创板/创业板更高) |
| 单日换手率 | ≥ 20% |
| 振幅 | ≥ 15% |
| 连续 3 日涨跌幅 | 累计 ≥ ±20% |

(具体标准随交易所规则调整,实操时以交易所最新公告为准)

**3. 量化席位的识别**

近年量化高频策略大量使用专门席位(常见的有华鑫上海、招商深圳总部、东方财富等)。
特征:
- 上榜频率极高(几乎每天都有)
- 单股买入金额相对均匀(无明显方向性)
- 风格跨题材、跨板块(没有明确擅长领域)

**意义**:量化席位上榜 ≠ 主力意图,**他们的买卖更多是程序化高频,跟随价值低**。要把量化席位从"游资跟踪"中剔除。

**4. 营业部 vs 总部账户**

- **传统游资**:用具体营业部账户(如"华泰证券股份有限公司宁波解放南路证券营业部")
- **机构 / 量化**:多用总部账户("XX 证券股份有限公司总部")
- **公募基金**:用"机构专用"席位

判断时优先看营业部级别的明细——总部账户和"机构专用"信息密度低很多。

**5. 北向资金与龙虎榜的关系**

北向资金(沪股通 / 深股通)有独立的"陆股通专用席位",上榜时显示为"沪股通专用席位"或"深股通专用席位":
- 北向席位大额买入 → 中线机构资金看好,但与短线节奏不同
- 北向席位连续大额卖出 → 该票中线吸引力下降,谨慎

**6. A 股具体应用场景对照**

- **打板族**:重点跟"赵老哥"类首板狙击席位,但要 T+1 卖
- **接力族**:重点跟"章盟主"类接力席位,持仓 2-3 天
- **波段族**:看"宁波桑田路"等多席位协同的强势题材
- **中线族**:重点关注"机构专用"席位的连续买入
- **新股族**:重点跟"炒新一族"等专注新股次新股席位
