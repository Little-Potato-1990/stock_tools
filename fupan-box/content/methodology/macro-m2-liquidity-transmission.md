## 核心命题

> "**水多鱼大**"——股市的长期表现,**根本上由流动性决定**。
> 但流动性不是简单的"M2 增速",而是**"钱从央行到股市"的传导链路**:**央行 → 商业银行 → 实体经济 → 居民 → 股市**,每一步都可能"截留"。
> M2 增速 ≠ 股市流动性 —— **关键看 M1 / M2 比例 + 居民储蓄变化 + 北上资金**。
> "**M2 增速 - GDP 增速 - CPI**" 是判断 A 股流动性的最有用指标。

## 一、问题的起源 —— 为什么"放水"不一定推动股市

很多散户的简单逻辑:"央行放水 → 股市必涨"。
但实际:
- 2015 年:央行降息降准 + 股市大涨(逻辑成立)
- 2019 年:央行降准 + 股市横盘(逻辑不成立)
- 2022-2023 年:央行多次降息 + 股市持续低迷(逻辑反向)
- 2024 年 9 月:央行降息 + 股市急涨(逻辑成立)

为什么"放水"有时管用,有时不管用?**因为流动性传导链路有多个环节,每一步都可能截留**。

理解这个传导链路,你就能:
- 提前判断"放水"会不会传导到股市
- 知道哪些指标真正影响股市流动性
- 避免被"央行降息"等表面信号误导

## 二、底层原理 —— 流动性传导的五步链路

钱从央行到股市,要经过 **5 个环节**:

```
央行 → 商业银行 → 实体经济 → 居民收入 → 居民储蓄 → 股市
  ①         ②           ③          ④          ⑤        
```

**环节 1:央行放水(基础货币)**
- 工具:降准、降息、再贷款、MLF / SLF / OMO
- 效果:增加基础货币 + 降低资金成本

**环节 2:商业银行扩张信贷(广义货币 M2)**
- 商业银行用低成本资金放贷给企业 / 居民
- M2 增速上升

**环节 3:钱进入实体经济**
- 企业贷款 → 投资 / 生产
- 居民贷款 → 买房 / 消费
- M1 增速上升(企业活期存款)

**环节 4:居民收入上升**
- 经济活跃 → 居民收入增加
- 居民有闲钱

**环节 5:居民储蓄进入股市**
- 居民选择把钱投入股市
- 股市流动性上升 → 股价上涨

**关键问题**:每一步都可能"**截留**":
- 商业银行不愿放贷(企业不愿借)→ 钱卡在银行(2018-2022 年的部分时段)
- 企业不愿投资 → 钱卡在企业账上(M1 增速不上)
- 居民不消费 / 不投资 → 钱卡在储蓄(2022-2024 年)
- 居民不进股市,而是买房 / 储蓄 → 股市没流动性

**核心结论**:**只看央行放水,看不到全貌——必须看完整传导链路**。

## 三、最重要的几个流动性指标

### 指标 1:M2 增速

**定义**:M2 = 流通中的现金 + 活期存款 + 定期存款 + 储蓄存款
**含义**:**广义货币供应量**,反映整体货币环境

**判断标准**:
- M2 增速 > GDP 增速 + CPI 增速 → 流动性宽松
- M2 增速 < GDP 增速 + CPI 增速 → 流动性紧张
- 中国正常 M2 增速:8-10%

**应用**:
- 看 M2 增速是否高于 GDP + CPI
- 看 M2 增速的变化趋势(上升 / 下降)

**局限**:
- M2 不能反映"钱在哪一环节"
- 高 M2 不一定流入股市

### 指标 2:M1 增速

**定义**:M1 = 流通中的现金 + 活期存款
**含义**:**狭义货币供应量**,反映企业的流动性

**判断标准**:
- M1 增速上升 → 企业活期存款多 → 企业准备投资 / 经营活跃
- M1 增速下降 → 企业活期存款少 → 企业不愿投资 / 经营萎缩

**应用**:
- M1 同比增速 > 5% → 经济活跃
- M1 同比增速 < 5% → 经济偏弱
- M1 转负 → 经济衰退信号

**M1 / M2 比例**:
- M1 / M2 比例上升 → 钱从定期变活期 → 投资 / 消费意愿强
- M1 / M2 比例下降 → 钱从活期变定期 → 储蓄意愿强(经济差)

**实战意义**:**M1 增速比 M2 增速更能反映"钱进不进股市"**。

### 指标 3:社融(社会融资规模)

**定义**:社融 = 实体经济从金融体系获得的全部资金
**含义**:**实体经济实际拿到的钱**

**判断标准**:
- 社融增速 > M2 增速 → 钱真的进了实体
- 社融增速 < M2 增速 → 钱卡在金融体系

**应用**:
- 关注**新增社融**(每月数据)
- 关注**社融结构**(贷款 / 债券 / 表外融资)

**实战意义**:社融反映"**实体经济是否真的在扩张**"。

### 指标 4:居民存款

**定义**:居民在银行的总存款
**含义**:**居民的"钱袋子"**

**判断标准**:
- 居民存款大幅增加 → 居民在储蓄(经济差 / 不敢消费)
- 居民存款大幅减少 → 居民在花钱 / 投资(经济好)
- **居民储蓄存款 / 居民总收入 > 30%** → 储蓄意愿强(经济偏弱)

**实战意义**:
- 大量居民储蓄是"**潜在流动性**"
- 一旦居民信心恢复,这些钱会流入股市 / 房市
- 例:2024.9 政策刺激后,大量储蓄涌入股市

### 指标 5:北上资金

**定义**:通过沪深港通流入 A 股的境外资金
**含义**:**外资对 A 股的态度**

**判断标准**:
- 北上持续净流入 → 外资看好 A 股
- 北上持续净流出 → 外资看空 A 股
- 极端净流入 / 流出 → 短期反弹 / 调整信号

**实战意义**:
- 北上资金是 A 股最敏感的"领先指标"之一
- 北上重仓股(白酒 / 医药 / 消费)对北上资金极敏感

## 四、A 股流动性的"最有用指标"

把上面这些指标综合,**A 股流动性最有用的判断公式**:

```
A 股流动性强度 = M2 增速 - GDP 增速 - CPI 增速
```

**判断**:
- > 5% = **极宽松**(流动性大幅过剩,股市有大牛市基础)
- 3-5% = 宽松(流动性宽松,股市有结构性机会)
- 0-3% = 中性(流动性适中,股市平衡)
- < 0 = 紧张(流动性紧张,股市承压)

**典型案例**:
- **2014-2015**:M2 增速 12%,GDP 7%,CPI 1.5% → 流动性强度 3.5% → 牛市
- **2018**:M2 增速 8%,GDP 6.5%,CPI 2% → 流动性强度 -0.5% → 熊市
- **2024 年中**:M2 增速 6.2%,GDP 5%,CPI 0.5% → 流动性强度 0.7% → 弱
- **2024.9 后**:政策强刺激,流动性预期改善 → 短期反弹

## 五、央行政策工具及其影响

### 工具 1:降息(影响利率)

**作用机制**:
- 降低基准利率 → 贷款成本下降 → 鼓励企业借钱 / 居民买房 / 投资
- 直接影响:**长债收益率下降 → 债市上涨 + 股市估值提升**

**对股市的影响**:
- 利好:**地产、金融、成长股**(降息环境下)
- 中性:消费、医药
- 利空:储蓄类(理财收益下降)

### 工具 2:降准(影响货币乘数)

**作用机制**:
- 降低存款准备金率 → 商业银行可放贷资金增加 → M2 增速提升

**对股市的影响**:
- 利好:**金融、地产、周期**
- 中性:其他

### 工具 3:再贷款 / 结构性工具

**作用机制**:
- 央行直接给商业银行低成本资金,定向支持某些行业(如科技、绿色、小微)

**对股市的影响**:
- 利好:被定向支持的行业(如 2024 年的科技 / 新能源)

### 工具 4:股市相关工具(2024 年新设)

**作用机制**:
- 央行设立"**股票回购增持再贷款**" → 上市公司可低成本融资回购自己股票
- 央行设立"**互换便利**" → 机构可向央行借资金购买股票

**对股市的影响**:
- 直接利好 A 股(2024.9 后大盘急涨的核心原因之一)

## 六、推导:为什么 A 股近年"放水不灵"

2018-2023 年央行多次放水,但 A 股长期低迷。原因(用流动性传导链路分析):

**环节 1:央行放水 ✓**
- 多次降息降准
- 基础货币增加

**环节 2:商业银行扩张信贷 ⚠️**
- M2 增速维持 8-10%
- 但**商业银行更愿意买国债,不愿放贷给企业**(企业风险高)

**环节 3:实体经济 ❌**
- 企业不愿借钱(经济预期差)
- 房地产去杠杆,地产链条收缩
- M1 增速持续低位甚至负增长

**环节 4:居民收入 ❌**
- 实体经济不扩张 → 居民收入增长慢
- 居民信心下降

**环节 5:居民储蓄进股市 ❌**
- 居民选择**储蓄**(2022-2024 年居民储蓄大幅上升)
- 居民选择**还房贷**(提前还贷潮)
- **不进股市**

**结果**:央行放水 → 钱卡在银行 → 没有进入股市 → 股市低迷。

**核心结论**:**单纯放水不能推动股市,必须打通整个传导链路**。

2024.9 政策的关键:**直接给股市补水**(回购贷款 + 互换便利),**绕过传导链路**——这就是为什么这次有效。

## 八、三个常见误区

**误区一:"央行放水股市必涨"**
错。要看是否能传导到股市。
**修正方法**:**看完整传导链路,而不是单一指标**。

**误区二:"M2 增速高就够了"**
错。M2 高 + 钱卡在银行 ≠ 股市流动性高。
**修正方法**:**结合 M1 / M2 比例 + 社融 + 居民储蓄**综合判断。

**误区三:"忽略北上资金"**
北上资金是 A 股最敏感的流动性指标之一。
**修正方法**:**每天关注北上资金动向**。

## 十、A 股本土化注解

**1. 中国货币政策的独立性**
中国货币政策与美联储**独立但有联动**:
- 美联储加息时,中国央行被迫维持利差(不能太激进降息)
- 美联储降息时,中国央行有更多政策空间

应对:**关注中美利差,影响中国货币政策空间**。

**2. 中国"储蓄率高"的影响**
中国居民储蓄率全球最高(40%+),这意味着:
- **巨大的潜在流动性**
- 一旦信心恢复,可流入股市
- 但储蓄率高也意味着"消费不足"

应对:**关注居民储蓄变化,可能预示流动性变化**。

**3. A 股具体流动性案例**

- **2014-2015**:降息降准 + 杠杆资金 → 流动性强度 3.5%+ → 大牛市
- **2018**:去杠杆 + 资管新规 → 流动性强度 -0.5% → 大熊市
- **2020**:疫情救市 + 全球放水 → 流动性强度 4%+ → 结构牛
- **2022-2024**:经济转型 + 流动性传导不畅 → 长期低迷
- **2024.9**:央行直接给股市补水 → 短期急涨

**4. 判断"放水能否传导到股市"的快速方法**

✅ **能传导**(看到这些信号):
- M1 增速回升
- 社融数据超预期
- 居民储蓄停止快速增长
- 北上资金转为净流入
- 央行直接给股市的工具

❌ **传导不畅**(看到这些信号):
- M1 增速持续低位 / 负增长
- 居民储蓄持续大幅增加
- 北上资金持续净流出
- 房地产数据持续低迷

## 十一、一句话记忆

> 流动性不是"央行放了多少水",**是"水从央行流到股市的过程"**。
> **关键看 M2 - GDP - CPI 的差值,以及 M1 / M2 比例 + 居民储蓄 + 北上资金**。
> 单纯央行放水 ≠ 股市必涨,**要看是否能打通传导链路**。
