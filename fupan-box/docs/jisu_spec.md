## 二、大盘情绪 - 指标全集

### 市场整体表现指标（OverviewBar 必须出现的行）
- 大盘成交金额（亿）
- 收盘涨停 / 收盘跌停（不含 ST）
- 开盘涨停 / 开盘跌停
- 开盘高开 / 开盘大低开
- 收盘上涨率 / 盘中上涨率
- 上证上涨率 / 深证上涨率 / 创业板上涨率

### 主板相关
- 主板 - 1板 / 2板 / 3板 / 4板 / 5板 / 6板 / >=7板
- 主板最高板
- 主板 - 妖股（带连板数）
- 主板上日涨停实体涨幅 / 平均涨幅 / 开盘平均

### 创业板相关
- 创业板 - 1板 / 2板 / >=3板
- 创业板 - 妖股（带连板数）
- 创业板上日涨停实体涨幅 / 平均涨幅 / 开盘平均

### 特殊指标
- 上日妖股上涨率 / 上日强势票上涨率 / 上日弱势票上涨率
- 一字板数量 / 反包板数量 / 炸板数量

### 强势&反包系数（独立小图）
- 妖股系数 / 强势系数 / 反包系数 / 前排效应

## 三、染色规则（重点!）

| 区域 | 是否染色 | 染色对象 |
|---|---|---|
| OverviewBar | 仅染**比率/百分比**类（上涨率、炸板率、涨停率、强势率、各类系数） | rate scale (红5档/绿4档) |
| 计数类（涨停数、跌停数、开盘高开数、成交金额…） | **不染色**，仅显示数字 | - |
| 连板天梯卡片 | 左边 3px 板级色条；徽章按板级染色 | board level color |
| 题材/行业 cell | 整格按 change_pct 渐变染色 | change scale |
| 强势股 cell | 按 change_pct 染色 | change scale |
| 资金前排 cell | 按金额或成交占比渐变 | count_orange scale |

**禁止**：所有表格/卡片不要 hover 改色；只有 click 跳详情才允许改色（背景或边框）。

## 四、字号 / 间距 / 边框（视觉规范）

```
--font-xs:  10px      (徽章、辅助文字)
--font-sm:  11px      (表格小文字)
--font-md:  12px      (主要数据)
--font-lg:  13px      (标题)
--font-xl:  16px      (页面/区块大标题)
```

- **行高**：表格 row 强制 `height: 32px`、`line-height: 1`
- **padding**：cell 0.35rem 0.5rem
- **border**：仅 `border-bottom: 1px solid var(--border-color)`，不要右侧描边
- **圆角**：卡片 4px，徽章 2~3px
- **卡片间距**：图表容器之间 8px (`space-y-2`)
- **背景**：3 层
  - 一级 `--bg-primary`（页面）
  - 二级 `--bg-secondary`（侧栏/表头）
  - 三级 `--bg-tertiary` 或 `--bg-card`（卡片）

## 五、子页面布局速查

### 5.1 连板天梯
- **连板天梯**：日期列网格（最近 6~8 天），每列从高板到低板降序排
- **相关个股查询**：选中一个股票后，右侧列出与其同题材/同板级的其他股票
- **筛选**：板级 ≥ N、是否一字、是否反包、题材关键词
- **缩略模式**：每个 cell 只显示「股票名 + 板数」，密度提升 3 倍

### 5.2 强势股
- 5 个市场 scope：近期强势 / 主板 / 创业板 / 科创 / 北交
- 每个 scope：日期列网格（7 列）
- 顶部筛选条：全部 / 高成交 / 低成交 / 一字板 / T字板 / 反包板

### 5.3 行业追踪
- **行业分析**：排名 × 日期热力网格（最近 7 天，每列 ≤ 50 行业）
- **筛选**：与强势股同样的 6 选项
- **连续强势行业**：单独 tab，黄底 + 红箭头标注 2 天以上持续高排名的行业

### 5.4 资金分析（4 sub-tab）
- 行业资金分析（净流入 × 日期）
- 题材资金分析（净流入 × 日期）
- 相关个股（drawer）
- **每日交易量前排标的**（按 amount 降序，每天 50 只）

### 5.5 龙虎榜分析（独立一级模块）
- 每日龙虎榜信息：交易个股 / 涨幅 / 买入金额 / 卖出金额 / 净额 / 营业部
- 营业部历史买卖记录：选定营业部，列出全部历史交易
- 游资追踪：知名游资席位（章盟主、宁波桑田等）的资金动向
